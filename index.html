About Me

James Smith, British entrepreneur and developer based in San Francisco.

I'm the co-founder of Bugsnag with Simon Maynard, and from 2009 to 2012 I led up the product team as CTO of Heyzap.

In the past I've built a few other web and facebook apps (wegot.tv, helpthirst.com, sponsor-me facebook app) and I've built a few popular open-source projects, which you can find on my Github page.

Follow @loopj

Projects

Bugsnag
Bugsnag captures exceptions in real-time from your web, mobile and desktop applications, helping you to understand and resolve them as fast as possible.
Gitscore
Web app that calculates your GitHub reputation score, based on your open-source contributions. Built entirely in Node.js.
Asynchronous Http Client for Android
An asynchronous callback-based HTTP client for Android used by Instagram, Heyzap, Trello and many other large apps.
jQuery Tokeninput
A jQuery plugin for selecting multiple items from a predefined list, using autocompletion as you type to find each item. Used by Twitter and many other large sites.
jQuery Simple Slider
An unobtrusive jQuery plugin for selecting a value from a numerical range by simply dragging a slider.
Smart Image View for Android
A drop-in replacement for Android’s standard ImageView which allows images to be loaded from URLs or the phone’s contact address book. Images are cached to memory and to disk for super fast loading.
